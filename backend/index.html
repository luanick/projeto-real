Biblioteca de Empréstimos

Projeto integrado desenvolvido para a disciplina de Teste de Software da Católica SC. Hospedagem estática via Cloudflare Pages conectada ao backend no Render.com.

Acessar Aplicativo no Render Código Fonte no GitHub

Descrição do Sistema

O sistema permite que leitores anunciem livros ou materiais físicos disponíveis para empréstimo em sua região, permitindo que outros membros da comunidade manifestem interesse nas obras e façam combinações diretas.

Focado nas melhores práticas de TDD (Test-Driven Development) e Shift-Left, cada funcionalidade crítica foi desenvolvida com garantia prévia de conformidade, eliminando regressões desde o primeiro commit.

Resumo da Matriz de Ferramentas

Comparação de ferramentas para cobertura de testes e entrega de qualidade contínua no projeto:

Ferramenta Camada Substitui Vitest + Supertest? Melhor Uso no Contexto
Vitest + Supertest Unitário/API Base do projeto Testar lógica pura de controladores, validações e rotas Express em SQLite.
Playwright / Cypress E2E / UI Não, complementa Simular interações reais de cadastro e solicitação de livros no navegador.
k6 Performance Não, complementa Verificar robustez e latência na listagem concorrente do catálogo de livros.
GitHub Actions Pipeline CI Não, complementa Automação completa da suíte de testes unitários e de integração a cada push.
